# 📋 Todo List App

A sleek and minimal **Todo App** built with 
**⚛️ React**, 
⚡ **Vite**, and 
💨 **Tailwind CSS**.

---

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as complete/incomplete
- 🗑️ Delete tasks
- 💻 Responsive design
- ⚡ Super fast with Vite
- 🎨 Styled with Tailwind CSS
